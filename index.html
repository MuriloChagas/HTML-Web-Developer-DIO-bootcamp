HTML é linguagem de programação?

voltar

É comum acreditar que o HTML é uma linguagem de programação, porém ele não é uma. O HTML é uma linguagem de marcação, pois ele formata como o texto aparecerá no front-end de um site por exemplo, mas para uma melhor experiência no site é preciso que mais elementos participem do codigo dessa página como o css, o javascript entre outros elementos.

Entendendo a diferença entre listas ordenadas e não ordenadas

voltar

Listas ordenadas são listas criadas no HTML que possuem uma ordem numérica. Ja as listas não ordenadas no HTML, diferente de como o nome diz, elas possuem uma ordem (uma sequencia), mas essa sequencia não é numerada. Elas podem ser utilizadas de diversar formas no corpo de uma pagina HTML, mas são mais utilizadas na criação de menus com o auxilio do css, para transforme a navegabilidade mais "confortavel".
